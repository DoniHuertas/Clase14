Backend con Node.js

Objetivos de la clase de hoy son:

  • Protocolo HTTP y HTTPS
  • Cliente con el servidor
  • Node.js
  • Librerias y paquetes
  • Terminal
  • Git/GitHub

Cuando realizamos un request, lo ralizamos usando un método o method. Estos pueden ser:

  • GET: Solicitar información
  • POST: Enviar información// Crear
  • PUT: Editar
  • DELETE: Eliminar

Dentro del response, viene incluido entre otras propiedas, un status Code o código de estado, referido a la operación que estemos realizando. Estos pueden ser:

    1xx: Códigos de información
    2xx: Operación exitosa.
    3xx: Redirección. Nos redirigen a una nueva URL
    4xx: Error del lado del cliente
    5xx: Error del lado del servidor

Node.js

Node es un entorno de ejecución de JS. Nos va a permitir escribir código de JS en nuestro backend.

Librerias de Node.js: Son funciones y métodos que se encuentran en un paquete y nos sirven para cuando necesitemos escribir nuestros códigos en diferentes programas.
